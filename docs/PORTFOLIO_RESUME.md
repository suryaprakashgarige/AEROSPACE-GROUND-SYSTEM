## 📄 ATS-Friendly Resume Description

### **Senior DevOps / SRE Engineer (Portfolio Project)**
**Satellite Telemetry Monitoring and Alert System (STMAS)**

- **Designed & Implemented** a ground station telemetry system to simulate, ingest, and monitor real-time aerospace satellite metrics (altitude, velocity, battery voltage) using **FastAPI (Python)**, **PostgreSQL**, and **WebSockets**.
- **Configured Containerized Environments** with **Docker** and **Docker Compose**, streamlining local orchestrations of databases, backend microservices, Prometheus servers, Grafana dashboards, and the ELK stack.
- **Architected Cloud Infrastructure** using **Terraform** modules to provision an AWS VPC, Security Groups, private subnets, RDS PostgreSQL instances, and an AWS EKS (Kubernetes) cluster.
- **Orchestrated Kubernetes Deployments** within EKS, writing declarative manifests for namespace, RBAC (ServiceAccounts, Roles), ConfigMaps, Secrets, ingress reverse proxies, and Horizontal Pod Autoscalers (HPA) targeting CPU utilization.
- **Enabled Observability & Logging** by integrating **Prometheus** endpoints, provisioning customized **Grafana** health dashboards, and configuring log ingestion streams using **Logstash**, **Elasticsearch**, and **Kibana**.
- **Constructed CI/CD Automation Pipelines** using **GitHub Actions** to automate code linting, run unit/integration test suites (pytest), perform vulnerability checks (Trivy), build and push Docker images, and execute rolling updates on EKS.

## 🔗 LinkedIn Portfolio Post

🚀 **Excited to share my latest aerospace DevOps project: Satellite Telemetry Monitoring & Alert System (STMAS)!**

I built a production-style aerospace ground station dashboard that simulates real-time satellite telemetry, monitors service health, and showcases a complete modern DevOps pipeline.

**Key features include:**
- **Real-time Telemetry Gateway**: FastAPI & WebSockets stream latitude/longitude ground tracks, battery levels, temperatures, and signal strength.
- **Infrastructure as Code**: Terraform modules provision AWS VPC, RDS Postgres, and AWS EKS.
- **Kubernetes Orchestration**: Zero-downtime rolling updates, resource limits, liveness/readiness probes, and CPU-based autoscaling (HPA).
- **Full Observability & Logs**: Prometheus scrapes metrics, Grafana visualizes host load, and the ELK Stack indexes service logs.
- **CI/CD Automation**: GitHub Actions runs security scans, test suites, builds Docker images, and deploys to EKS.
